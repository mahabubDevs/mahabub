    I am Md Mahabub, a passionate and dedicated MERN stack web developer currently pursuing my BSc in Computer Science and Engineering (CSE). My primary focus is on building high-quality, scalable, and dynamic web applications using MongoDB, Express, React, and Node.js. I have over 3 years of experience in web development, which has equipped me with the skills to handle both frontend and backend challenges.
    In the frontend, I specialize in React.js and Next.js, crafting seamless user experiences with efficient state management through Redux. I am also proficient in Tailwind CSS and Bootstrap, which enable me to create responsive and visually appealing designs.
    On the backend, I have strong experience in Node.js and Express, developing robust RESTful APIs that integrate smoothly with databases. My database management skills include MongoDB and Redis, allowing me to optimize data storage and retrieval for complex applications.
    I have a deep understanding of React Router DOM for building dynamic web applications and am continuously improving my skills in advanced database design and TypeScript. As a developer, I enjoy solving challenging problems and staying updated with the latest trends and best practices in the industry. My goal is to deliver solutions that not only meet client expectations but exceed them, with a keen focus on performance, scalability, and user experience.
